# PIX Code Agent - Setup & Installation Guide

This guide explains how to install, configure, and launch the PIX Code Agent using the unified automated setup scripts.

---

## ⚡ Quick Start

Choose your operating system below to complete the full installation automatically:

### Windows
1. Double-click the [setup.bat](file:///c:/Users/MohamedThoufeeq/Desktop/pix-code-agent/setup.bat) file in your file explorer, **or** open PowerShell / Command Prompt and run:
   ```cmd
   setup.bat
   ```
2. Enter your `PIX_API_KEY` when prompted (you can retrieve this key from positka.net).
3. Once completed, launch the agent with:
   ```cmd
   python trigger.py
   ```

### Linux / macOS
1. Open your terminal and navigate to the project directory.
2. Run the cross-platform setup script:
   ```bash
   bash setup.bat
   ```
3. Enter your `PIX_API_KEY` when prompted.
4. Once completed, launch the agent with:
   ```bash
   python trigger.py
   ```

---

## ⚙️ What the Installer Does Under the Hood

The [setup.bat](file:///c:/Users/MohamedThoufeeq/Desktop/pix-code-agent/setup.bat) script automates the complete onboarding workflow:

1. **System Prerequisite Checks**:
   - Verifies if **Python 3.11+** is installed. If missing, it uses `winget` (Windows) or system package managers (Linux/macOS) to install it automatically.
   - Verifies if **Node.js 18+** is installed. If missing, it installs the LTS version automatically.
2. **Backend Setup**:
   - Creates a Python virtual environment (`.venv`) inside the `backend` directory.
   - Upgrades `pip` and installs all dependencies listed in `requirements.txt`.
3. **Environment Setup**:
   - Creates a `backend/.env` file from the template `backend/.env.example`.
   - Prompts you interactively for your `PIX_API_KEY` and configures it automatically.
4. **Frontend Setup**:
   - Installs React and TypeScript package dependencies in the `frontend` folder using `npm install`.

## 🚀 Launching the Application

To launch both the **FastAPI Backend** and the **Vite React Frontend** simultaneously in separate terminal windows, run the following command in the root folder:

```bash
python trigger.py
```

- **Frontend Access**: The web-based developer console will open at `http://localhost:5173`.
- **Backend API Access**: The API server is hosted at `http://localhost:8000` (docs available at `/docs`).

## 👤 Customizable Profile & AI Quotes

Once inside the PIX Code Agent web dashboard, you can customize your workspace identity:

- **Custom Username**: Click on the profile card at the bottom-left corner of the sidebar to type in your name. This will update the text and the avatar's initial dynamically.
- **Hourly AI Quotes**: The profile card renders a motivational programming quote that changes automatically every hour. Quotes are generated by the PIX LLM API (falling back to a local database of programming wisdom if offline).

## 🛠️ Manual Troubleshooting

If you encounter issues running the automated setup script, you can perform the steps manually:

### 1. Manual Backend Installation
```bash
cd backend
python -m venv .venv

# Activate Virtual Environment:
# On Windows (cmd):
.venv\Scripts\activate.bat
# On Windows (PowerShell):
.venv\Scripts\Activate.ps1
# On Linux/macOS:
source .venv/bin/activate

# Install requirements
pip install -r requirements.txt

# Copy and configure key
cp .env.example .env
# Open .env and replace PIX_API_KEY=pix_xxx with your API key
```

### 2. Manual Frontend Installation
```bash
cd frontend
npm install
npm run dev
```
